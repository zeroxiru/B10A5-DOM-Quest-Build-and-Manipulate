IMPORTANT FAQ

What is the Document OBject Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (such as an HTML or XML file) as a tree of objects, where each object corresponds to a part of the document, like elements, attributes, and text.

Key Points:

  • Hierarchical Structure: The DOM organizes the document in a tree-like structure, with each node representing an element, attribute, or piece of text.
  • Dynamic Interaction: It allows scripts (often JavaScript) to access, manipulate, and update the content, structure, and style of a document dynamically.
  • Platform and Language Independent: The DOM can be used with various programming languages and is not limited to a specific platform.
How do you select an element from the DOM?
  1. document.getElementById('id'): Selects an element by its unique ID.
  2. document.querySelector('selector'): Selects the first element that matches a CSS selector.
  3. document.querySelectorAll('selector'): Selects all elements that match a CSS selector (returns a NodeList).
  4. document.getElementsByClassName('class'): Selects all elements with a specific class name.
  5. document.getElementsByTagName('tag'): Selects all elements with a specific tag name.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is an efficient way to handle events by attaching a single event listener to a parent element rather than adding listeners to individual child elements. This approach leverages the .target property of the event object to identify which specific child element triggered the event, allowing the parent to handle it appropriately.

How do you manipulate an element's attributes and styles using the DOM?

To change an element's attributes and styles in JavaScript, first get the element using methods like getElementById or querySelector. Then, use setAttribute to change its attributes (e.g., change a link with element.setAttribute('href', 'new-link')). You can also modify its inline CSS by using the style property (e.g., element.style.color = 'orange' to change text color).
